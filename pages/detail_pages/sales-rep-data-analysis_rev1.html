Introduction:¶

This report will investigate a sales representative and revenue dataset.

This dataset includes the following properties:

      Date, SalesRep, Region, Product, Revenue, Color, Units, Revenue

This investigation will focus on revenue generated, time periods of productivity, and most popular products sold.

Over the course of this report, the following questions will be explored:

  • Question 1: What is the total and % revenue per region?
  • Question 2: What is the total revenue per year?
  • Question 3: For each Sales Rep, what are their total revenues per region?
  • Question 4: Which Sales Reps have the largest % of revenue for each of the regions?
  • Question 5: Which quarter has the highest earnings?
  • Question 6: What is the revenue per month for each year?
  • Question 7: What are the top 3 products sold?

# Function converts month number to month name. 
def month_num_2_name(num):
    month_dict = {
        1:"January", 2: "February", 3: "March",
        4: "April", 5: "May", 6: "June",
        7: "July", 8: "August", 9: "September",
        10: "October", 11: "November", 12: "December"
    }
    
    return month_dict[num]

# Function will generate the proper quarter based on the observed month.
def generate_quarter(month):
    if month in ["January","February","March"]:
        return "Q1"
    elif month in ["April","May","June"]:
        return "Q2" 
    elif month in ["July","August","September"]:
        return "Q3"
    elif month in ["October","November","December"]:
        return "Q4" 
    else:
        return "Error: Name of month is not valid"
In [24]:
